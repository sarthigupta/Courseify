# Courseify

> Convert any YouTube playlist into a structured, Udemy-like learning experience.

## Tech Stack
- **Frontend**: React + Vite, Tailwind CSS, shadcn/ui, Zustand, React Router, Axios
- **Backend**: Node.js, Express.js, Zod, JWT, Google OAuth
- **Database**: MongoDB (Mongoose)
- **External**: YouTube Data API v3

## Quick Start

### 1. Clone & Install
```bash
# Backend
cd backend && npm install

# Frontend
cd frontend && npm install
```

### 2. Environment Variables
Copy `.env.example` files in both `backend/` and `frontend/` and fill in your values.

### 3. Run
```bash
# Backend (port 5000)
cd backend && npm run dev

# Frontend (port 5173)
cd frontend && npm run dev
```
